[Tomoki Ota's Blog](https://tomomon-blog.pages.dev/) のリポジトリ。

## Usage

Dev-Containerを立ち上げて、コンテナ内で以下のコマンドを叩く。

```sh
make run
```

OR

```sh
bun run dev
```

OR

```sh
bunx --bun vite
```

http://localhost:3001/ (もしくは、自分で指定したポート) を開く
